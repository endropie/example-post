# Development kit REST API with Lumen

## How to install

1. Install package

```bash
composer install
```

2. Create .env file

3. Key generate

```bash
php artisan key:generate
```

4. JWT secret

```bash
php artisan jwt:secret
```

5. Migrate database

```bash
php artisan migrate:fresh --seed
```

6. Run app

```bash
sh run.sh
```

And enjoy!
